# Standard Stars 
WIP for Anna Madeleine



# Mechanics
- Plate views
	- randomly placed on a point on a sphere of radius 0.5 meters.
	- could be placed in a circle instead to avoid people craning their necks etc.
- Hot-cold targeting system
